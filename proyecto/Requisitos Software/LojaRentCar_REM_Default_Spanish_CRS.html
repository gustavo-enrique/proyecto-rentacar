3 Descripción Actual del Sistema

Actualmente para realizar la reservación de un vehículo se debe hacer de forma presencial, la empresa maneja todos los procesos administrativos a mano con poca eficiencia para tener un registro de todos sus vehículos.

Entrada de nuevos competidores en el ámbito laboral.
Competidores usan sistemas web externos que puede facilitar el uso a todos los clientes.
Debido a la pandemia, no existe variedad de vehículos.
Crisis económica por el Covid-19.
Los clientes optan por permanecer dentro de sus casas.

El sistema para reservación y renta de vehículos cuenta con un:
Servidor donde guardara la información generada.

    • Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando El usuario pueda realizar el pago de ingresando los datos de la tarjeta de crédito.
    Precondición 1. Que el usuario haya seleccionado la opción Reservar.
    2. Que se haya cargado correctamente la ventana Ingresar Tarjeta.
    Secuencia normal Paso Acción
    1 El actor Cliente (ACT-0001) selecciona la opción Reservar.
    2 El sistema Carga la ventana Ingresar Tarjeta.
    3 El actor Cliente (ACT-0001) ingresa los datos correspondientes para realizar el pago con tarjeta(N° de Tarjeta, Caducidad, Código de Seguridad/CVV2, Titular de la Tarjeta, Dirección, ciudad, Provincia y Código Postal).
    4 El sistema Verifica los datos.
    5 El sistema Verifica que la cuenta tenga fondos suficientes.
    6 El sistema Envía un correo de confirmación.
    7 El sistema Muestra el mensaje ?La reserve se ha realizado con éxito?
    8 El sistema Carga la ventana Ficha de Reserva
    Postcondición 1. Se mostrar un mensaje de confirmación.
    2. El sistema cargará la ventana Ficha de Reserva.
